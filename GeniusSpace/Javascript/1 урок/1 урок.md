1. Змінні (Variables) — "Коробки для даних"
   const — (constant) створює коробку, вміст якої не можна повністю замінити іншою коробкою.

let — створює коробку, в яку можна покласти нове значення пізніше.

= — знак присвоєння. Це команда: "Візьми те, що справа, і поклади в те, що зліва".

2. Рядки (Strings) — "Текст у лапках"
   Приклад: const message = "Welcome to Bahamas!";

" " або ' ' — лапки кажуть комп'ютеру: "Це просто текст, не шукай тут команди".

.length — (довжина) властивість. Пишеться через крапку після назви змінної. Рахує кількість усіх знаків, включаючи пробіли.

.toUpperCase() — метод. Робить текст ВЕЛИКИМ.

() — обов'язкові дужки в кінці означають: "Виконай цю дію прямо зараз".

3. Об'єкти (Objects) — "Шафа з підписаними полицями"
   Приклад: const user = { name: "Ivan" };

{ } — фігурні дужки створюють об'єкт.

name — це ключ (ярлик на полиці).

"Ivan" — це значення (те, що лежить на полиці).

: — двокрапка розділяє ярлик і саму річ.

, — кома розділяє різні полиці в одній шафі.

Маніпуляції з об'єктом:
user.age = 25 — через крапку ми додаємо нову полицю.

delete user.city — команда delete викидає полицю з шафи.

user["like flowers"] — квадратні дужки та лапки потрібні, якщо в назві ярлика є пробіл.

true / false — (Boolean) логічний тип. true — так, false — ні.

4. Цикл for...in — "Автоматичний перебір"
   Це спосіб змусити комп'ютер самому прочитати всі ярлики в шафі.

JavaScript
for (let key in user) {
console.log(key + ": " + user[key]);
}
for — команда "повторюй дії".

let key — ми створюємо тимчасовий ярлик key. На кожному колі в нього записується назва наступної полиці.

in — вказує, в якій саме шафі (user) ми шукаємо.

user[key] — ми кажемо: "Знайди в шафі user полицю, назва якої зараз лежить у key, і покажи, що там всередині".

- — склеює назву ярлика і його вміст в один красивий рядок для консолі.

5. Вивід у консоль
   console.log() — команда "виведи результат на екран (у панель розробника)". Все, що в дужках, ми побачимо в браузері.

Порада: Збережи цей текст. Це твій фундамент.

Конспект по дз:

1. Змінні та копіювання
   JavaScript
   const name = "Іван";
   const city = name;
   const — створює незмінну "коробку".

name = "Іван" — поклали текст у коробку.

city = name — скопіювали вміст із name у city. Тепер в обох "Іван".

2. Шаблонні рядки (Бектики)
   `привіт ${1}` — виведе привіт 1.

`привіт ${"name"}` — виведе привіт name (бо в дужках текст у лапках).

`привіт ${name}` — виведе привіт Іван (бо бере значення зі змінної).

${ } — знак, що дозволяє вставити код або змінну всередину тексту.

3. Видобування чисел (Parsing)
   JavaScript
   parseInt("13cvb"); // 13 (шукає ціле число на початку)
   parseFloat("12.9sxd"); // 12.9 (шукає число з крапкою)
   typeof numA; // "number" (перевіряє тип даних)
   parseInt — "вирізає" ціле число з рядка, поки не зустріне літеру.

parseFloat — те саме, але зберігає дробову частину (після крапки).

4. Математика та округлення
   0.1 + 0.2 — через особливості пам'яті комп'ютера дає 0.30000000000000004.

.toFixed(1) — залишає лише 1 знак після крапки. Перетворює число на рядок "0.3".

5. Об'єкт Math
   Math.max(20, 10, 50) — знаходить найбільше число.

Math.random() — генерує випадкове число від 0 до 0.99.

Math.floor() — округлює число донизу (відкидає хвіст).

6. Робота з текстом (String)
   .length — властивість. Рахує кількість символів.

.toUpperCase() — метод. Робить усі літери ВЕЛИКИМИ.

7. Об'єкти (Objects)
   JavaScript
   const user = {};
   user["like flowers"] = true;
   delete user.city;
   {} — пустий об'єкт ("шафа").

user.name = "Ivan" — створили полицю name.

delete — видаляє полицю разом із вмістом.

[" "] — квадратні дужки потрібні, якщо в назві ключа є пробіл.

8. Цикл for...in
   JavaScript
   for (let key in user) {
   console.log(key + ": " + user[key]);
   }
   for — початок циклу.

let key — тимчасова змінна для назви ключа.

in — вказує на об'єкт.

user[key] — дістає значення, що лежить під цим ключем.
